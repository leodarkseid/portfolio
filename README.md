# Badmus Kolade - Portfolio

A lightweight, performant personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Project Structure

- `index.html`: Main HTML entry point.
- `css/`: Stylesheets.
- `js/`: Application logic (navigation, form handling).
- `assets/`: Images and resources.

## Running Locally

To view the portfolio locally, you can use any static file server.

Python 3:
```bash
python3 -m http.server 8085
```

Then open [http://localhost:8085](http://localhost:8085).
